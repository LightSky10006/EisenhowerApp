## 特色

### 核心功能
- **四象限待辦清單**：基於艾森豪矩陣的重要/緊急分類系統
- **雙視圖模式**：清單視圖與平面座標視圖，可透過 BottomNavigationBar 切換
- **任務完整資訊**：每個任務包含標題（72字限制）和備註（400字限制）
- **座標系統**：任務以（重要程度、緊急程度）權重顯示於座標圖上

### 互動體驗
- **點擊新增**：點擊象限空白處或座標圖空白處即可新增待辦
- **拖拽編輯**：點擊任務點可直接編輯任務內容
- **權重調整**：每個任務可自訂「重要」與「緊急」權重（-5~+5），並自動依象限限制範圍
- **座標操作**：平面座標支援縮放、平移和雙擊重置

### 視覺化與主題
- **平面座標分布**：GeoGebra 風格的專業座標系統，包含網格、標籤和軸線
- **多主題支援**：淺色、深色、跟隨系統、賽博朋克2077（螢光黃/藍/紫）
- **主題一致性**：所有主題色彩、座標軸、座標點、標籤皆自動同步主題
- **自適應 UI**：四象限自動填滿可用空間，適應不同螢幕尺寸

### 本地化與設定
- **雙語支援**：繁體中文/English 介面切換
- **數據持久化**：使用 SQLite 本地儲存，支援任務的完整 CRUD 操作
- **設定頁面**：獨立的設定頁面，方便調整主題、語言等偏好

## 如何使用

### 安裝與執行
1. 安裝 [Flutter](https://flutter.dev/docs/get-started/install)
2. 下載本專案
3. 在專案資料夾執行：
   ```bash
   flutter pub get
   flutter run
   ```

### 基本操作
- **新增任務**：在清單視圖中點擊象限，或在座標視圖中點擊空白處
- **編輯任務**：點擊任務項目或座標點，可修改標題、備註、重要程度和緊急程度
- **刪除任務**：在編輯對話框中點擊刪除按鈕
- **切換視圖**：使用底部導航欄在清單視圖和座標視圖間切換

### 進階功能
- **座標縮放**：在座標視圖中使用雙指縮放或滑鼠滾輪
- **座標平移**：拖拽座標圖進行平移
- **重置視圖**：雙擊座標圖重置縮放和位置
- **主題切換**：在設定頁面選擇偏好的主題風格
- **語言切換**：在設定頁面選擇中文或英文介面

## 主要檔案

### 專案結構
```
lib/
├── main.dart                      # 主程式，包含 UI、主題、語言切換邏輯
├── models/
│   ├── task.dart                  # 任務資料模型（包含標題、備註、座標等）
│   └── app_enums.dart             # 應用程式枚舉定義
├── screens/
│   ├── eisenhower_list_screen.dart # 四象限清單視圖頁面
│   ├── quadrant_plane_screen.dart  # 平面座標視圖頁面
│   └── settings_screen.dart        # 設定頁面
├── widgets/
│   └── quadrant_painter.dart       # 自訂座標繪圖組件
└── services/
    └── database_service.dart       # SQLite 資料庫服務
```

### 資料庫結構
- 使用 SQLite 本地儲存
- 支援任務的完整 CRUD 操作
- 自動處理資料庫版本升級和 schema 變更

### 主題系統
- 支援四種主題：淺色、深色、跟隨系統、賽博朋克
- 所有 UI 元素自動適配主題色彩
- 賽博朋克主題採用螢光黃/藍/紫配色方案

## 主題系統

### 可用主題
- **淺色主題**：清爽的白色背景，適合日間使用
- **深色主題**：護眼的深色背景，適合夜間使用
- **跟隨系統**：自動根據裝置系統設定切換
- **賽博朋克主題**：螢光黃/藍/紫配色，呈現未來科技感

### 主題特色
- BottomNavigationBar 顏色隨主題自動變化
- 座標軸、網格、標籤、任務點顏色完全同步
- 賽博朋克主題採用專業的螢光色彩搭配
- 所有對話框、按鈕、文字顏色自動適配
